# Cognitive Manifold 2025

This repository contains the full research preprint and supporting dataset for:

**Detecting Stable Cognitive Manifolds in Human-AI Dialogue (2025)**  
Author: Shawn Scheer

This project investigates cross-model convergence in LLM descriptions of a single participant's reasoning geometry.  
It includes:

- `index.html` — The full formatted research paper  
- `/transcripts` — Raw conversation logs used for analysis  
- All supplementary data required for Zenodo DOI archival

A permanent DOI will be generated through Zenodo upon release.
